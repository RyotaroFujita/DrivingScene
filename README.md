# DrivingScene
対応機種：Windows 64bit

アプリケーションの実行方法：

  ①同一フォルダに以下のファイル，フォルダを配置
	
    VheicleSense_BurstDebugInformation_DoNotShip
    VheicleSense_Data
    MonoBleedingEdge
    VheicleSense.exe
    UnityCrashHandler64.exe
    UnityPlayer.dll
		
  ②VheicleSense.exeを実行

自動車の操作方法:十字キー

アプリケーションの終了方法：Alt+F4
